# 5/3/1

Personal Wendler 5/3/1 gym-tracking PWA. Vanilla JS, no build step, works offline after first load.

## Run locally

```
python3 -m http.server 8000
```

Then open http://localhost:8000.

## Deploy

Hosted via GitHub Pages on the `main` branch.

## Install on iPhone

Open the deployed URL in Safari → Share → Add to Home Screen.
